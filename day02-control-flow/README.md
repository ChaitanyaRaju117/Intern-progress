# Day 2 — Control Flow & Functions

Learning if/elif/else, loops, functions, scope and docstrings.
Also compared 4 free LLMs on the same prompt.

## Files
- `fizzbuzz.py` — FizzBuzz using if/elif/else and for loop
- `prime_checker.py` — Prime number checker with functions
- `loops_practice.py` — Loops, break, continue, scope, default args
- `llm_comparison.md` — Comparison of Claude, ChatGPT, Gemini, Mistral

## How to run

```bash
python fizzbuzz.py
python prime_checker.py
python loops_practice.py
```
